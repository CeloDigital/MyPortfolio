# Marcelo.tech — Portfólio Profissional

Landing page ultra-moderna, responsiva e de alta performance desenvolvida para apresentar meus serviços de Desenvolvimento Web Full Stack e suporte especializado em TI/Hardware.

---

## 🚀 Tecnologias Utilizadas

* **React.js** — Estrutura da aplicação e componentização eficiente.
* **Tailwind CSS** — Estilização moderna baseada em utilitários com foco em performance e modo escuro nativo.
* **Lucide React** — Pacote de ícones minimalistas e otimizados.
* **Vite** — Build tool extremamente rápido para o desenvolvimento front-end.

---

## ✨ Funcionalidades e Melhorias Implementadas

* **Design Unificado (Deep Dark)**: Interface construída inteiramente sobre uma paleta de cores escura e consistente, evitando quebras visuais entre seções.
* **Grid Assimétrico de Serviços**: Layout adaptável que destaca os serviços principais através de proporções inteligentes e efeitos visuais sutis.
* **Contador de Status Animado**: Faixa de dados com efeito JavaScript nativo que realiza a contagem numérica incremental assim que entra na tela do usuário.
* **Componente de Perfil HUD**: Avatar em formato circular envolto por anéis tecnológicos dinâmicos com animações de pulso, rotação e linhas de scanner estilo HUD.
* **Formulário de Contato Inteligente**: Micro-interações focadas em campos, validação de e-mail em tempo real e feedback visual estruturado (estados de carregamento e sucesso).
* **Integração de Redes Sociais**: Botões de conexão direta via SVG nativo injetado, prevenindo problemas de quebra de pacotes externos.
